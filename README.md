## Task 3 – Water Jug Problem using Depth First Search

Implementation of the classical Artificial Intelligence search problem using the Depth First Search algorithm.

### Objectives

* Simulate two water jugs with different capacities
* Achieve a target amount of water
* Use DFS to explore possible states
* Display the solution path once the goal state is reached

### Concepts Applied

State Space Search, DFS Algorithm, Backtracking

## Task 4 – N-Queen Problem

A program designed to solve the N-Queen puzzle using recursion and backtracking techniques.

### Description

The objective of this problem is to place N queens on an N×N chessboard such that no queen threatens another.

### Concepts Used

Backtracking, Recursion, Constraint Satisfaction Problem
